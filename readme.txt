User access permission management plugin by role
== Description ==
Plugin features :
User role management
Managing access to menus in such a way that if the menu is not accessed, the screen will be locked
Post type support
Support for plugins and template menus
completely free

== Installation ==
This section describes how to install the plugin and get it working.
e.g.
1. Upload `manage-user-access-permission.zip` to the `/wp-content/plugins/` directory
2. Unzip the plugin
3. Activate the plugin through the 'Plugins' menu in WordPress
4. In the users menu, the manage user access option is enabled
